Hostel Management System

The Hostel Management System (HMS) is a project that implements fundamental CRUD operations. It allows users to Create, Read, Update, and Delete data stored in a locally-hosted database using PHP and MySQL. The system efficiently manages student data, room assignments, and other hostel-related tasks, showcasing the integration of backend development with a user-friendly interface.

Tic Tac Toe - Game

This Tic Tac Toe game is developed using HTML, CSS, and JavaScript, focusing on front-end technologies. It features an interactive user interface where two players can compete. The project demonstrates dynamic DOM manipulation and the use of JavaScript logic to check for winning conditions and handle gameplay.
